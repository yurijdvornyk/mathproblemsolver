Open source, easy to use solution for working with mathematical problems

MathProblemSolver is the desktop application designed to simplify the process of creating and calculating different mathematical problems. It provides the user with friendly interface and simple API that allows to concentrate on mathematical aspects of the project.

MathProblemSolver business process consists of 6 major steps:

  1. Download MathProblemSolver application and ProblemLibrary. You can get sources and binaries by clicking on one of the buttons above.

  2. Create your problem. Extend Problem class or implement IProblem interface, override the methods, implement problem solving and build the project. MathProblemSolver uses for this purposes Problem API, which was designed specially for you!:) It's realy simple, but also powerful. More about Problem API you can read here.

  3. Load problem in MathProblemSolver. Open "Problem Repository" () and add your problem. If it has some external dependencies, add them in "External Libraries Repository" ().

  4. Set input data. Select your problem in Problem Data menu, fill all parameter values that your problem need to be solved.

  5. Solve problem. Click on the "Solve" button () to start calculating the problem result.

  6. Get results. After the problem is solved, check its results and save them to popular file formats or just print them.
