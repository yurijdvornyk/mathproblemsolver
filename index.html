Welcome to MathProblemSolver page

  1. Download MathProblemSolver application and ProblemLibrary.

  2. Create your problem. Extend Problem class or IProblem interface, override the methods, implement problem solving and build the project.

  3. Load problem in MathProblemSolver. Open "Problem Repository" and add your problem. If it has some external dependencies, add them in "External Libraries Repository".

  4. Set input data. Select your problem in Problem Data menu, fill all parameter values that your problem need to be solved.

  5. Solve problem. Click on the "Solve" button to start calculating the problem result.

  6. Get results. After the problem is solved, check its results and save them to popular file formats or just print them.
